# Screenshots for README

Save your screenshots here with these exact filenames:

| Filename | What to capture |
|----------|-----------------|
| `needs-board.png` | Homepage — public needs board with org cards |
| `donate-form.png` | Donation form — step 1 or 2 of the donate flow |
| `coordinator-dashboard.png` | Coordinator dashboard — overview or donations pipeline |

Then run `git add docs/screenshots/*.png` and commit. The README will display them automatically.
